## How to Use

1. Open `index.html` in your web browser
2. Enter the coefficients a, b, and c
3. Click "Solve" to see the solution

## Types of Solutions

- **Discriminant > 0**: Two distinct real solutions
- **Discriminant = 0**: One repeated real solution
- **Discriminant < 0**: Two complex conjugate solutions
